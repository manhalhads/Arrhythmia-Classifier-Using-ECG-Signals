# To run the notebook:

1. Download the .ipynb file.
2. Open it in Google Colab.
3. Ensure cells are run sequentially from top to bottom.
4. From the Runtime menu, select Change runtime type, then choose GPU as the hardware accelerator.

# To edit the latex file:

1. Download the zip folder.
2. Open Overleaf and make a new project.
3. Upload the zip folder.
4. Edit deliverable_2.tex.
